# dotfile
whole dotfile for global config command line tool

## Setup Vim Plug
```
$ cp -r .vim $HOME/
```

## Setup .bashrc or .zshrc
```
alias vim=mvim -v
```

## Install Macvim
```
$ brew install macvim
```

## Install Vim Plugin
```shell
$ vim +PlugInstall
```

## Clean Vim Plugin
```shell
$ vim +PlugClean
```
